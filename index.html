HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want on the outside of an element.
  • A padding indicates how much space we want on the inside of an element.
  • Inline CSS: is coded directly within the HTML file. Each CSS attribute is applied to each HTML element. Less common since if there should be a style change, it would require you to go back and edit each element, individually.
  • Internal style sheet: CSS attributes are imbedded in the HTML file header. This allows for CSS rules to be applied generally to certain types of HTML elements.
  • External style sheet: CSS is coded in a separate file, then linked to the HTML file once completed. The CSS file is linked within the HTML header. This is the most commonly used type of CSS, since all of the styling is kept separate, making it easier to change when necessary.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JS uses conditional logic. This can be coded by using if/else statements. The if block code will run only if the condition is met.
  • Loops are used to iterate an object.
  • Functions are sets of instructions to tell the computer to do a task. Functions are written, but then need to be called in order to run.
